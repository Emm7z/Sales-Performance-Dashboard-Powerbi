# Sales Performance Analytics Dashboard

## Project Overview
This project provides fast, intuitive analysis of sales performance across categories, regions, and time periods. The goal was to replace manual reporting with a self-service dashboard for quick insights.

## Challenge
The sales team had a large dataset but lacked a fast way to visualize performance, making weekly reporting time-consuming and error-prone.

## Solution
Developed an **interactive Power BI dashboard** with data modeling, charts, slicers, and timelines. The dashboard allows users to filter by category, region, and date range with one click.

## Results
- Delivered a self-service KPI dashboard that **eliminated manual weekly reporting**  
- Reduced reporting time by 100%  
- Enabled the sales team to focus on strategy and improve sales performance
